== Description ==

Horizontal Slider with scroll is a simple yet powerful slider plugin which is unique the way it works. As the name suggests, you can slide the different images from left to right or right to left with mouse's scroll wheel. You can create slider with existing or new images or can use custom post types to make a responsive slideshow. The plugin comes with a default shortcode which you can use on any page where you want this slider.

New Feature Update: Now you can swap the slideshow on mobile too with version 1.1

= Key features of this plugin include =

* Horizontal slider with images only.
* Horizontal Slider with Custom Post Types like products, blog posts etc.
* Ability to show Post Title.
* Ability to show or hide Post Description.
* Ability to show or hide Post Terms.
* Ability to show or hide Post Read More Button.
* Ability to control the number of posts.
* Ability to set image dimensions of the slider.
* Ability to control Post Description word size.
* Ability to show Post Title, Description, Terms on Mouse Hover.


== Installation ==

The plugin is simple to install:

 * Download horizontal-scroll-slider.1.0.zip
 * Unzip
 * Upload the plugin directory to your /wp-content/plugins directory
 * Go to the plugin menu page and activate the plugin.

== Frequently asked questions ==

= How to use Horizontal Slider with Scroll plugin? =

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Then go to new post or page and enable classic editor, you can paste the default shortcode i.e. [HSS].


= What is the benefit of using this plugin? =

User can easily change page or post text into uppercase or lowercase or capitalize or sentence case or invert case etc..
